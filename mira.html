Présentation du projet Mira : The Last Hope


Ce projet a été réaliser en équipe de 10 sur 2 semaines.

Nous avions comme consigne de créer un Platformers 2D dans un univers Solarpunk.

Nous étions 4 a travailler sur le code du jeux.

J'ai travailler sur le moteur fournis pas l'intevenant afin de transformer les entitées gerer en rectangle (au lieu de cercle).

J'ai aussi travailler sur le game design du jeu avec 3 autres personnes du groupe.

Sur le gameplay du jeu j'ai réaliser l'éditeur de niveau, ainsi que plusieurs mécaniques.
